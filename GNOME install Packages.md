# Описание приложений гном

1. **Baobab -** утилита анализа использования дискового пространства для среды рабочего стола GNOME. Она помогает пользователям визуализировать использование дискового пространства и определять крупные файлы или каталоги.

2. **Epiphany -** веб-браузер для среды рабочего стола GNOME. Он стремится быть простым и эффективным, обеспечивая при этом современный опыт веб-просмотра.

3. **Evince -** просмотрщик документов для среды рабочего стола GNOME. Он поддерживает несколько форматов документов, таких как PDF, PostScript и различные форматы изображений.

4. **GDM -** GNOME Display Manager. Это менеджер входа для среды рабочего стола GNOME, который предоставляет графический экран входа для пользователей для аутентификации и запуска своих сеансов.

5. **gnome-backgrounds -** это пакет, содержащий коллекцию фоновых изображений для среды рабочего стола GNOME. Эти изображения могут использоваться в качестве обоев для настройки внешнего вида рабочего стола.

6. **gnome-calculator -** это приложение-калькулятор, разработанное для среды рабочего стола GNOME. Оно предоставляет базовые и научные функции калькулятора.

7. **gnome-calendar -** это приложение календаря для среды рабочего стола GNOME. Оно позволяет пользователям создавать, просматривать и управлять своими событиями и встречами.

8. **gnome-characters -** приложение для работы с символами и эмодзи в среде рабочего стола GNOME. Оно позволяет пользователям легко находить и вставлять различные символы в свои документы или сообщения.

9. **gnome-clocks -** приложение с часами и таймером для среды рабочего стола GNOME. Оно предоставляет возможность отслеживать текущее время, создавать таймеры и будильники.

10. **gnome-color-manager -** инструмент управления цветом для среды рабочего стола GNOME. Он предоставляет пользователю возможность калибровать цветовой профиль своего монитора для точного отображения цветов.

11. **gnome-connections -** приложение для управления соединениями и сетями в среде рабочего стола GNOME. Оно позволяет пользователям настраивать и управлять подключениями к Wi-Fi, VPN и другим сетевым службам.

12. **gnome-console -** приложение терминала для среды рабочего стола GNOME. Оно предоставляет пользователю доступ к командной строке для выполнения различных задач и команд в системе.

13. **gnome-contacts -** приложение для управления контактами в среде рабочего стола GNOME. Оно позволяет пользователям хранить, просматривать и управлять списком контактов, включая данные о телефонных номерах, адресах электронной почты и т. д.

14. **gnome-control-center -** центр управления настройками для среды рабочего стола GNOME. Он предоставляет пользователю доступ к различным системным настройкам, таким как сеть, звук, экран и т. д.

15. **gnome-disk-utility -** утилита дисков для среды рабочего стола GNOME. Она предоставляет пользователю инструменты для управления и анализа дисковых устройств, включая создание и форматирование разделов, а также проверку состояния дисков.

16. **gnome-font-viewer -** приложение для просмотра шрифтов в среде рабочего стола GNOME. Оно позволяет пользователям просматривать и устанавливать различные шрифты на своей системе.

17. **gnome-keyring -** менеджер ключей для среды рабочего стола GNOME. Он предоставляет безопасное хранение паролей, ключей шифрования и других конфиденциальных данных, а также автоматическое вход в систему без необходимости повторного ввода паролей.

18. **gnome-logs -** приложение просмотра журналов для среды рабочего стола GNOME. Оно позволяет пользователям просматривать системные журналы для отслеживания различных событий и ошибок в операционной системе.

19. **gnome-maps -** приложение карт для среды рабочего стола GNOME. Оно предоставляет пользователю доступ к интерактивным картам, позволяя искать местоположения, прокладывать маршруты, просматривать данные о транспорте и находить различные объекты и достопримечательности. Gnome Maps также может интегрироваться с другими сервисами карт, такими как OpenStreetMap, для обеспечения актуальных данных о местоположениях и маршрутах.

20. **gnome-menus -** инфраструктура для работы с меню в среде рабочего стола GNOME. Она обеспечивает создание и управление меню приложений и системных настроек.

21. **gnome-music -** приложение для прослушивания музыки в среде рабочего стола GNOME. Оно позволяет пользователям организовывать и воспроизводить аудиофайлы с их компьютера или подключенных устройств.

22. **gnome-remote-desktop -** инструмент удаленного доступа для среды рабочего стола GNOME. Он позволяет пользователям подключаться к удаленным компьютерам и управлять ими через сеть.

23. **gnome-session -** это компонент среды рабочего стола GNOME, который управляет сеансами пользователей. Он запускает и завершает сеансы рабочего стола, обеспечивая стабильную и безопасную среду работы.

24. **gnome-settings-daemon -** это фоновый процесс, отвечающий за управление настройками среды рабочего стола GNOME. Он обрабатывает изменения в настройках системы, таких как разрешение экрана, громкость звука и темы оформления.

25. **gnome-shell -** оболочка рабочего стола для среды рабочего стола GNOME. Она предоставляет пользователю интерфейс для взаимодействия с операционной системой, включая запуск приложений, управление окнами и доступ к системным функциям.

26. **gnome-shell-extensions -** набор расширений для оболочки рабочего стола GNOME. Они добавляют дополнительные функции и возможности к стандартному интерфейсу GNOME.

27. **gnome-software -** центр приложений для среды рабочего стола GNOME. Он предоставляет пользователю доступ к каталогу программного обеспечения, а также управление установленными приложениями и обновлениями.

28. **gnome-system-monitor -** приложение для мониторинга системы в среде рабочего стола GNOME. Оно позволяет пользователям отслеживать использование ресурсов процессора, памяти, дискового пространства и сети, а также управлять процессами.

29. **gnome-text-editor -** текстовый редактор для среды рабочего стола GNOME. Он предоставляет базовые функции редактирования текста, такие как вставка, удаление, поиск и замена.

30. **gnome-tour -** приложение, предназначенное для проведения пользователей по основным функциям и возможностям среды рабочего стола GNOME. Оно помогает новым пользователям освоиться с интерфейсом и функционалом системы.

31. **gnome-user-docs -** документация для пользователей среды рабочего стола GNOME. Она содержит руководства и справочные материалы по использованию различных функций и приложений GNOME.

32. **gnome-user-share -** это компонент среды рабочего стола GNOME, который обеспечивает возможность обмена файлами и папками через локальную сеть. Он позволяет пользователям легко делиться файлами с другими устройствами в их домашней сети.

33. **gnome-weather -** приложение погоды для среды рабочего стола GNOME. Оно предоставляет пользователю информацию о текущей погоде и прогноз на несколько дней для выбранного местоположения.

34. **grilo-plugins -** набор плагинов для Grilo, фреймворка для управления медиаконтентом в среде рабочего стола GNOME. Они добавляют поддержку различных сервисов потокового видео, аудио и фотографий.

35. **gufs -** утилита для доступа к файловой системе iPhone и iPod Touch в среде рабочего стола GNOME. Она позволяет пользователям обмениваться файлами между своим компьютером и устройствами Apple.

36. **gufs-afc -** модуль для управления файловой системой Apple File Conduit (AFC) в среде рабочего стола GNOME. Он обеспечивает доступ к файлам и данным на устройствах iPhone и iPod Touch через протокол AFC.

37. **gufs-goa -** модуль для интеграции с онлайн-аккаунтами (GOA) в среде рабочего стола GNOME. Он позволяет пользователям подключаться и управлять своими онлайн-сервисами, такими как Google, Facebook и др., непосредственно из окружения GNOME.

38. **gufs-google -** модуль для интеграции с Google-аккаунтами в среде рабочего стола GNOME. Он обеспечивает доступ к сервисам Google, таким как Google Drive и Google Calendar, из рабочего стола GNOME.

39. **gufs-gphoto2 -** модуль для работы с цифровыми фотокамерами через протокол gphoto2 в среде рабочего стола GNOME. Он позволяет пользователям импортировать фотографии с цифровых камер и управлять фотоколлекциями.

40. **gufs-ntp -** модуль для интеграции с сетевым временным протоколом (NTP) в среде рабочего стола GNOME. Он обеспечивает синхронизацию времени на компьютере с общедоступными временными серверами.

41. **gufs-nfs -** модуль для поддержки протокола сетевой файловой системы (NFS) в среде рабочего стола GNOME. Он позволяет пользователям монтировать удаленные файловые системы, расположенные на серверах NFS, на своем компьютере.

42. **gufs-onedrive -** модуль для интеграции с облачным хранилищем OneDrive в среде рабочего стола GNOME. Он обеспечивает доступ к файлам и папкам, хранящимся на OneDrive, из рабочего стола GNOME.

43. **gufs-smb -** модуль для поддержки протокола сетевого доступа к файлам (SMB) в среде рабочего стола GNOME. Он позволяет пользователям монтировать удаленные сетевые ресурсы, такие как общие папки Windows, на своем компьютере.

44. **loupe -** инструмент для увеличения и анализа изображений в среде рабочего стола GNOME. Он предоставляет возможность пользователю увеличивать и рассматривать изображения с высоким уровнем детализации.

45. **malcontent -** модуль для среды рабочего стола GNOME, который обеспечивает контроль доступа к нежелательному контенту. Он позволяет администраторам настраивать правила фильтрации контента, такие как веб-сайты, приложения или другие ресурсы, для повышения безопасности и управления доступом.

46. **Nautilus -** файловый менеджер для среды рабочего стола GNOME. Он предоставляет пользователю удобный интерфейс для работы с файлами и папками, включая возможность копирования, перемещения и удаления файлов, а также просмотра и редактирования их свойств.

47. **Orca -** экранный диктор и управляющий инструмент для среды рабочего стола GNOME. Он предоставляет доступ к компьютеру для людей с ограниченными возможностями зрения, обеспечивая озвучивание содержимого экрана и возможность управления интерфейсом с помощью голосовых команд или клавиатуры.

48. **rygel -** сервис мультимедийного контента для среды рабочего стола GNOME. Он позволяет пользователям передавать мультимедийные файлы, такие как музыка, фотографии и видео, с компьютера на другие устройства в локальной сети, поддерживающие протоколы DLNA и UPnP.

49. **simple-scan -** приложение для сканирования документов и изображений в среде рабочего стола GNOME. Оно предоставляет простой и интуитивно понятный интерфейс для сканирования документов с поддержкой различных типов сканеров.

50. **snapshot -** инструмент для создания и управления снимками экрана в среде рабочего стола GNOME. Он позволяет пользователям создавать снимки экрана целиком или только выбранных областей, а также выполнять различные операции с снимками, такие как сохранение, копирование или печать.

51. **sushi -** предварительный просмотр файлов в среде рабочего стола GNOME. Он позволяет пользователям быстро просматривать содержимое файлов, таких как текстовые документы, изображения или видео, прямо из окна файлового менеджера Nautilus, не открывая их в отдельных приложениях.

52. **tecla -** утилита для доступности и управления в среде рабочего стола GNOME. Она предоставляет пользователю возможность управлять компьютером с помощью адаптивных технологий, таких как альтернативные клавиатуры или устройства ввода с ограниченными возможностями.

53. **totem -** медиаплеер для среды рабочего стола GNOME. Он предоставляет возможность воспроизведения аудио и видео файлов различных форматов, а также поддерживает функции плейлистов и управления воспроизведением.

54. **tracker3-miners -** это набор инструментов для индексации и поиска контента в среде рабочего стола GNOME. Он обеспечивает интеграцию с системой отслеживания (Tracker), которая позволяет пользователям быстро находить файлы и данные на своем компьютере.

55. **xdg-desktop-portal-gnome -** интеграционный компонент для среды рабочего стола GNOME, который обеспечивает поддержку портала рабочего стола для приложений Flatpak. Он предоставляет интерфейс для взаимодействия с рабочим столом и другими приложениями для выполнения действий, таких как открытие файлов или запрос разрешений.

56. **xdg-user-dirs-gtk -** инструмент для управления стандартными папками пользователя в среде рабочего стола GNOME. Он позволяет пользователям настраивать и персонализировать папки, такие как Документы, Загрузки, Изображения и другие.

57. **yelp -** справочная система для среды рабочего стола GNOME. Она предоставляет доступ к руководствам пользователя и справочным материалам, а также инструкции по использованию различных функций и приложений GNOME.
